== Greek Restaurant Theme  ==

 Greek Restaurant is a lightweight, responsive Twitter bootstrap theme for restaurants, barbecues, pizzerias and more, ideal for mobile, pc and tablet devices. Some of it's features includes custom menus, header, customizer panel, full width pages, shortcodes and a fully featured Restaurant Reservation System that allows you to accept reservations from your customers. It's seo friendly as it uses semantic seo coding (roles - microdata etc).
